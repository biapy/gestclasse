Installation de Gest'classe_v7_plus ... étapes 1, 2 et 3

Attention : Si vous utilisiez une autre version de Gest'classe, n'oubliez pas de sauvegarder les données présentes dans les tables, avant de commencer l'installation.

Je n'ai pas prévu d'utilitaire, pour la migration vers la version v7_plus.
Pour éviter des problèmes pour la réinsertion des données, je vous invite à modifier la structure des tables manuellement par comparaison avec le fichier : table.sql

 

Les installations en local et sur Internet sont indépendantes.
Une fois les deux installations réalisées, il est très facile de transférer les données d'une installation à l'autre, en utilisant les utilitaires de Gest'classe.
L'installation en local commentée ici, correspond à une installation sous windows avec le logiciel easyphp.

LOCALEMENT

SUR INTERNET
Etape 1
Installez easyphp
Trouvez un hébergeur proposant une base de données Mysql
Etape 2
Copiez le dossier Gestclasse_v7_plus dans le dossier
C:\Program Files\EasyPHP1-8\www
Transférez par ftp le dossier Gestclasse_v7_plus chez votre hébergeur

Etape 3
Si tout s'est bien déroulé, vous devez maintenant consulter ce fichier avec votre navigateur ( Mozilla, i.e., ... ) en utilisant ce chemin :

Il faut tout d'abord lancez easyphp, un e doit alors clignoter en bas à droite de votre ordinateur . Puis, cliquez sur le e avec le bouton droit de la souris et sélectionnez Web Local . Ensuite, cliquez sur Gest'classe_v7_plus.
Le chemin est alors :
http://localhost/gestclasse_v7_plus/lisez_moi.htm
ou une adresse du type http://pc181031439119/gestclasse_v7_plus/installation/lisez_moi.htm
http://www.mon_herbergeur.truc/gestclasse_v7_plus/lisez_moi.htm

Poursuivre l'installation ...
