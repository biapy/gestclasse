Site dynamique pédagogique destiné aux enseignants - Mise en ligne de cours, d'exercices ... - Cahier de texte en ligne - Gestion des fiches des élèves - Trombinoscope des classes - Carnet de notes en ligne, calcul des statistiques, histogrammes - Agenda - Gestion des tables - Espace élève sécurisé - Espace prof sécurisé ...

Généralités :

Le programme comporte trois espaces distincts :
- un espace commun ( la page d'accueil, les liens, le plan du site, les rubriques générées automatiquement, les cahiers de texte, l'agenda )
- un espace prof ( sécurisé ) permettant la gestion du site, la gestion des cahiers de texte, la gestion des élèves ( une fiche pour chaque élève ), la gestion des contrôles et des notes, le calcul des statistiques ...
- Un espace élève ( sécurisé ) permettant à chaque élève ( ou aux parents ) de consulter une zone personalisée. Dans cette zone, l'élève peut consulter ses notes, ses moyennes, les statistiques des contrôles, les statistiques de la classe, les histogrammes générés automatiquement, un message du professeur, le trombinoscope de sa classe et compléter ou modifier sa fiche.

Les espaces protégés par mot de passe utilisent les sessions ( il n'est pas utile de ressaisir son mot de passe pendant la session ).

Installation :

Consultez le fichier lisez_moi.htm

Description de l'espace commun :
  • "La page d'accueil" que vous construisez à partir de l'epace prof : la mise à jour, votre mail, le compteur ( un simple fichier texte : cpt.txt ), trois zones dont la zone de droite que vous pouvez générer automatiquement
  • La rubrique "plan" générée automatiquement
  • La rubique "lien" générée automatiquement
  • La partie commune de l'agenda
  • Les différentes rubriques et sous rubiques et contenus que vous pouvez générer automatiquement
  • Le cahier de texte des classes
  • Il est possible de restreindre ou supprimer l'accès à certaines rubriques à partir de l'espace prof.

Description des différentes rubriques de l'espace prof :
Remarques préliminaires :
- La croix rouge signifie suppression ( définitive ... attention)
- Les fiches des élèves sont accessibles à partir de la plupart des rubriques
- Pour le contenu des pages, vous pouvez entrer du contenu HTML, du texte ou utiliser les styles de Gest'classe.

  • "Configuartion"
    Cette rubrique permet :
    - de modifier le design
    - de créer la première page
    - de modifier Le bas de page commun aux pages
    - de mofifier la mise à jour et votre mail
    - de modifier les restrictions
    - de saisir le mot de passe du professeur
    - de modifier le compteur
    - ...
  • "Rubrique"
    Utilisez cette rubrique pour ajouter des rubiques et des sous-rubriques et modifier l'ordre d'apparition.
  • "Les rubriques et sous-rubriques"
    Entrez dans une rubrique pour modifier le contenu ( haut et bas de page, création des divisions dans la page, contenu automatique ... )
  • "Liens"
    Création automatique des liens classés par division.
  • "Ajouter une classe"
    Utilisez cette rubrique pour entrer ou supprimer vos classes dans la base
  • "Ajouter un élève"
    Cette rubrique permet d'ajouter un élève dans la base de données (Il faut d'abord créer la classe).
    Par défaut le login de l'élève est son nom et le mot de passe est son prénom ( attention aux homonymes ! )
    Vous pouvez modifier les mots de passe par défaut dans la rubrique "Les fiches des élèves".
  • "Ajouter un devoir"
    Il suffit de compléter ... n'oubliez pas d'indiquer le trimestre et le coefficient.
    Après avoir cliqué sur "ajouter", les élèves apparaissent, et il suffit de compléter ( 99 pour indiquer q'un élève est absent ).
    Ensuite, vous pouvez modifier éventuellement les informations sur le devoir et surtout ( !!!! ) calculer les statistiques du devoir et les moyennes des élèves.
  • "Carnet de notes"
    Dans cette rubrique, vous pouvez consulter les notes, les moyennes des élèves, les statistiques de la classe, les statistiques des contrôles, les appréciations sur les élèves ( à remplir dans les fiches ) et les histogrammes.
    Pour modifier les informations d'un devoir, cliquez sur "modifier".
  • "Trombinoscope"
    Dans cette rubrique, vous pouvez consulter les trombinoscopes des classes après avoir déposé les photos dans le dossier "admilux/photo_classe" ( utilisez l'identifiant de l'élève pour nommer la photo ex : 34.jpg )
    Cliquez sur la photo pour voir la photo de l'èlève.
  • "Les fiches des élèves"
    Les classiques fiches des élèves ( qu'ils peuvent remplir à partir de leur espace )
    Le professeur peut remplir deux rubriques supplémentaires : "L'appréciation" et "Commentaire du professeur"
  • "Les dernières connexions "
    Cette rubrique permet d'être tenu au courant des dernières connexions des élèves dans leur espace
  • "Les dernières fiches modifiées par les élèves "
    Cette rubrique permet d'être tenu au courant des dernières modifications des fiches
  • "Cahier de texte" et "Archives du cahier de texte"
    Choisissez un jour, puis le type d'information ... faites quelques essais. Vous pouvez archiver les jours ...
    La rubrique "bloc-notes" permet d'entrer des informations sur la classe pour le professeur.
  • "Agenda"
    Un agenda pour le prof, un agenda pour le site et un agenda pour les classes ...
    Par défaut l'agenda commence à la date du jour que vous pouvez modifier de façon temporaire.
    Vous pouvez choisir les dates de vacances et modifier la date de fin ( stockée dans la table agenda )
    En cliquant sur un jour, vous pouvez remplir l'agenda ... ( la flèche verte indique que le message apparaît dans l'agenda de l'espace commun, la flèche rouge indique que le message apparaît dans l'agenda de la classe sélectionnées )
  • "Sauvegarder les tables"
    Vous pouvez sauvegarder les tables au format sql et au format csv . Le ( ou les ) fichier(s) est sauvegardé dans le dossier admilux/tables/ ...
  • "Exécuter des requêtes sur la base"
    Utilisez cette rubrique pour, par exemple, restaurer vos tables ...
  • "Vider les tables"
    Utilisez cette rubrique pour faire le ménage ... ( attention )
  • La sauvegarde des tables est fondamentale
Description des différentes rubriques de l'espace élève :
  • En se connectant, l'élève peut consulter un message du professeur.
  • "Voir et modifier mes informations"
    Dans cette rubrique l'élève peut modifier sa fiche
  • "Voir mes notes"
    Dans cette rubrique l'élève peut :
    - Consulter ses notes
    - Consulter ses moyennes
    - Consulter les statistiques des devoirs et les histogrammes
    - Consulter les statistiques de la classe et les histogrammes
  • "Trombinoscope"
    Dans cette rubrique, l'élève peut consulter le trombinoscope de sa classe
  • Il est possible de restreindre ou supprimer l'accès à certaines rubriques à partir de l'espace prof.
